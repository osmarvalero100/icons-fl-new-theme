Usar con html

Para usar estos iconos con HTML puro se debe poner el tag svg y dentro de este tag agregar el elemento use el cual debe tener como valor del atributo "id" el nombre del icono que desea mostrar.

Ejemplos:

                    
                        <svg>
                            <use xlink:href="#si-address1"></use>
                        </svg>
                    
                
                    
                        <svg>
                            <use xlink:href="#si-bulb"></use>
                        </svg>
